# Navigation

## Mobile (&lt; md / 768px)
- `AppHeader`: compact, safe-area-top, hamburger → dropdown nav
- `sm` (640px+): header right shows settings / admin / logout
- Home: `NavCard` 5-column shortcut grid
- Touch targets 44px minimum

## Tablet & desktop (≥ md)
- `AppSidebar`: fixed left, logo + icon nav + user actions at bottom
- No hamburger; no `AppHeader` on screen
- Home `NavCard` hidden; use sidebar for primary navigation
- Optional bottom nav: not implemented
